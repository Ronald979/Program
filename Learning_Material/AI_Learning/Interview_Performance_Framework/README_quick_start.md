# 快速開始（今天就可以做）

## 一、先決定三個目標
1. 一個可交付專案（例如本地 LLM 部署）
2. 一個可量化題庫（DSA/SQL）
3. 一套可講的 STAR 案例

## 二、今天要做的事（60 分鐘）
- 建立今日進度：`templates/notes/daily_log_template.md`
- 練習 1 題 SQL
- 練習 1 題 DSA
- 寫 150 字專案進度（含問題與解法）
- 5 分鐘錄音「你做這個專案的重點」

## 三、每週交付
- `README`/腳本至少更新 1 次
- 技術題：最少 6 題
- 專案日誌：至少 3 次 postmortem

## 四、每次練習都要回答三問
- 我這個題目的目標是什麼？
- 我怎麼驗證正確？
- 有什麼可改進的版本？
