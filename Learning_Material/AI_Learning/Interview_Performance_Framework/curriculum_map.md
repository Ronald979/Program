# 學習地圖（面試加分版）

## 核心主題
- `Foundation`
  - Python：例外處理、模組化、基本工程流程
  - C：記憶體、指標、結構體、`volatile/static/const`、編譯期觀念
  - Git 流程、commit 觀念、README 寫作
- `Algorithms`
  - 陣列、字串、雜湊、排序、DFS/BFS、DP 2 類常見題
  - 會估算複雜度與資源
- `Data`
  - SQL 查詢（join/window/CTE）
  - 資料清理、資料品質檢核
- `Embedded Systems / Firmware`
  - 啟動流程、中斷/例外、驅動基本抽象、任務時序
  - 線上 debug、看門狗、功耗/記憶體限制
- `ML/LLM`
  - LLM 本地部署、量化、推論瓶頸、成本控制
  - 資料流水線、訓練/微調指令與回灌
- `FPGA`
  - RTL 到位元流流程、時序收斂、資源分配
  - 模組化、board bring-up、硬體驗證
- `其他 AI 模型`
  - Embedding（句向量）與向量檢索（RAG 基石）
  - Transformer 小模型（DistilBERT / MiniLM）
  - CV 小模型（YOLO/ResNet/ViT 小版）
  - 語音模型（Whisper/Vosk/Wav2Vec2 small）
  - 推薦系統模型（LightFM / Factorization Machine / CatBoost）
- `System Design`
  - API 架構、佇列、快取、降載、監控
  - 可擴充架構設計與故障回復
- `Communication`
  - STAR 案例（2-3 個）
  - 專案成果敘事：問題→方案→指標→改進
