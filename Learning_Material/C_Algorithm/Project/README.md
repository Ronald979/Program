# C Algorithm Projects

本目錄對應 FPGA 的 `Project/`。每個子目錄都是一個可實作、可測試、可驗收的 C project。

固定子目錄：

```text
src/        C source implementation
include/    public headers
test/       unit/integration tests
docs/       Q&A, notes, learning log, gate review
build/      build output
tools/      helper scripts or data generators
benchmark/ benchmark cases and reports
```
