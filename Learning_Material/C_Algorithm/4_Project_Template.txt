  main.c
  parser.c
  buffer.c
  algorithm.c

  buffer.h
  algorithm.h

  test_buffer.c

## 9. Memory Ownership

請說明：
- 哪些 memory 由 caller 分配
- 哪些 memory 由 module 分配
- 誰負責 free
- 是否可能 memory leak
- 是否可能 dangling pointer
- 是否需要 deep copy

## 10. Error Handling

請設計：
- return code
- enum error type
- NULL pointer check
- invalid input handling
- buffer overflow handling
- file error handling

## 11. Test Case 設計

每個 Project 至少包含：
- normal case
- boundary case
- invalid input
- empty input
- large input
- memory error case
- repeated operation case

## 12. Debug Checklist

請包含：
- gdb 如何檢查

- valgrind 如何使用
- 如何定位 segmentation fault
- 如何檢查 memory leak
- 如何檢查 off-by-one error

## 13. Performance Measurement

如果適用，請包含：
- latency
- throughput
- memory usage
- time complexity
- space complexity
- benchmark 方法
