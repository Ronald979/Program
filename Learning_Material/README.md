# Learning Material

這個資料夾是我和 Codex 深度合作的主要工作區。

用途：

- 學習素材
- 實作練習
- 筆記
- Q&A
- debug 紀錄
- gate review
- 尚未整理成履歷專案的實驗內容

目前內容：

| Folder | Purpose |
|---|---|
| `AI_Learning` | 面試框架、學習方法、筆記模板、面試演練素材 |
| `C_Algorithm` | C 語言、資料結構、演算法、C Linux 前置訓練與專案素材 |
| `FPGA` | FPGA、Verilog、Zynq、AXI、INT8 accelerator 學習與專案素材 |

## 工作規則

1. 所有未完成學習都先放在這裡。
2. 每個 project 要先有 README、Q&A、Learning Notes、Gate Review。
3. 只有完成驗收的 project 才能移到 `CV` 裡變成履歷敘述。
4. 學習狀態與下一步任務不要寫在這裡，統一寫到 `Learning_Roadmap`。

## 和 CV 的關係

這裡是「學習和實作過程」。

`CV` 是「完成後的履歷敘述與 Project PPT 素材」。

當某個 project 完成後，更新順序是：

1. 在這裡完成 code、測試、筆記與 gate review。
2. 到 `Learning_Roadmap` 更新完成狀態。
3. 到 `CV/PERSONAL_PROJECTS_INDEX.md` 更新履歷摘要。
4. 到 `CV/PROJECT_PRESENTATION_NOTES.md` 更新 PPT 展開內容。
