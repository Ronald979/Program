# 學習地圖（面試加分版）

## 核心主題
- `Foundation`
  - Python 函式、例外處理、模組化
  - Git 流程、commit 觀念、README 寫作
- `Algorithms`
  - 陣列、字串、雜湊、排序、DFS/BFS、DP 2 類常見題
  - 會用複雜度估算
- `Data`
  - SQL 查詢（join/window/CTE）
  - 資料清理、資料品質檢核
- `ML/LLM`
  - LLM 本地部署、量化、推論瓶頸、成本控制
  - 資料流水線、訓練指令與回灌
- `其他 AI 模型`
  - Embedding（句向量）與向量檢索（RAG 基石）
  - Transformer 小模型（DistilBERT / MiniLM）
  - CV 小模型（YOLO/ResNet/ViT 小版）
  - 語音模型（Whisper/Vosk/Wav2Vec2 small）
  - 推薦系統模型（LightFM / Factorization Machine / CatBoost）
- `System Design`
  - API 架構、佇列、快取、降載、監控
  - 簡單可擴充架構設計
- `Communication`
  - STAR 案例（2-3 個）
  - 專案成果敘事：問題→方案→指標→改進
