# Interview Performance Framework

這個 Folder 用來集中管理「不是只學 LLM，而是讓你在面試更亮眼」的整體流程。

## 目標
- 讓學習從「散」變成「流程化」
- 以你目前條件（GTX 1050、本機推理優先）實際可執行
- 打造可以反覆複用的面試作品、題解與回顧機制

## 目錄
- `01_Foundation`：程式與基礎數學底盤
- `02_Algorithm_System`：資料結構、演算法、SQL、軟體基本能力
- `03_MLOps_Project`：專案化學習、LLM 以外的 AI 產品能力
- `04_Systems_Design`：系統設計與架構能力（LLM 相關場景可擴展）
- `05_Interview_Practice`：模擬面試、行為題、簡報與作品包
- `templates`：可直接複用的記錄與交付模板

## 核心主張
- 你不用再「只補缺」；你要建立**可反覆執行的機制**。
- 每個主題都對應：
  - `輸入`（你要學什麼）
  - `實作`（你要做什麼）
  - `輸出`（你要交付什麼）
  - `回顧`（你如何改進）
