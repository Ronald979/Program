# Learning Roadmap

這個資料夾是未來每次打開專案時，第一個要看的地方。

用途：

- 目前進度
- 未攻克問題
- 接下來任務
- 已完成任務
- Project 是否已經整理成 CV
- Project 是否已經製作 PPT
- 每週/每階段復盤

## 建議閱讀順序

1. `01_CURRENT_STATUS.md`
2. `02_PROJECT_PROGRESS_TRACKER.md`
3. `03_ROADMAP_60_DAYS.md`
4. `04_LEARNING_RETROSPECTIVE.md`

## 三個核心問題

每次回來學習時，先回答：

1. 目前進度如何？還有什麼問題沒被攻克？
2. 接下來還有什麼任務還沒完成？
3. 已經完成了什麼？是否已經整理成 CV？是否已經製作 PPT？

## Folder Relationship

| Folder | Role |
|---|---|
| `Learning_Material` | 學習、筆記、實作、debug、gate review |
| `CV` | 完成後的履歷敘述、個人 project 摘要、Project PPT 素材 |
| `Learning_Roadmap` | 進度追蹤、下一步、復盤、完成度 |
| `AI_Agent_Monitor` | AI Agent 專用監視架構，任務開始/結束時更新整體狀態 |
